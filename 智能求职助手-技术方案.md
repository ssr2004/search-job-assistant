# 智能求职助手 — 技术方案

## 一、项目概述

### 1.1 项目背景

参考开源项目 [interview-agent](https://github.com/lhb-zero/interview-agent)（Java + Spring Boot + Vue），用 Python + React 重新实现并扩展为完整的"求职助手"产品。

interview-agent 实现了面试题生成 + RAG 检索增强问答，有成熟的四阶段 RAG 检索流水线和 RAGAS 评估体系。我们在其基础上：
1. 用 Python + React 重新实现（展示多语言能力）
2. 扩展简历筛选功能（匹配算法设计）
3. 扩展技能差距分析功能（自动识别候选人与 JD 的差距）
4. 形成从"面试准备"到"简历投递"的完整求职助手

### 1.2 项目定位

一个面向求职者的智能助手平台，覆盖求职全流程：
- **面试准备**：输入岗位方向 → AI 生成面试题 + 知识点
- **知识问答**：基于 RAG 的智能问答，支持混合检索 + Re-ranking
- **简历筛选**：上传简历 + JD → 自动匹配评分 → 生成筛选报告
- **技能差距**：自动识别候选人与 JD 的技能差距 → 生成学习推荐

### 1.3 为什么不像玩具

| 维度 | 玩具项目 | 本项目 |
|------|---------|--------|
| RAG 架构 | 简单向量检索 | 四阶段流水线（查询改写 → 混合检索 → Re-ranking → LLM） |
| 评估方式 | "感觉还行" | RAGAS 四指标（Faithfulness / Context Precision / Recall / Relevancy） |
| 匹配算法 | 调 LLM 就完事 | 多维度算法（硬性条件 + 语义匹配 + 综合评分） |
| 流式输出 | 一次性返回 | SSE 打字机式输出 + Markdown 实时渲染 |
| 工程深度 | 单一流程 | 混合检索 + RRF 融合 + Cross-Encoder 精排 |

### 1.4 简历包装话术

> **智能求职助手** | Python / FastAPI / React / LangChain
> - 实现四阶段 RAG 检索流水线（查询改写 → 混合检索（向量+关键词 RRF 融合）→ Cross-Encoder Re-ranking 精排 → LLM 生成），基于 RAGAS 框架评估，Faithfulness 达 0.87

> - 实现 SSE 流式响应 + Tool Calling，支持 LLM 自主调用知识库检索工具
> - 构建技能差距分析模块，自动识别候选人与 JD 的技能差距并生成个性化学习推荐

```python
class DocumentParser:
    """文档解析器"""
    
    def parse(self, file_path: str) -> list[dict]:
        """解析文档，返回分块列表"""
        # 1. 格式识别 + 文本提取
        file_type = self.detect_type(file_path)
        if file_type == "pdf":
            raw_text = self.parse_pdf(file_path)
        elif file_type == "md":
            raw_text = self.parse_markdown(file_path)
        elif file_type == "txt":
            raw_text = self.parse_txt(file_path)
        
        # 2. 文本清洗
        cleaned = self.clean_text(raw_text)
        
        # 3. 智能分块
        chunks = self.smart_split(cleaned)
        
        return chunks
    
    def clean_text(self, text: str) -> str:
        """文本清洗（参考 interview-agent 的 DocumentTextCleaner）"""
        # 去除页眉页脚
        text = re.sub(r'第\s*\d+\s*页.*?\n', '', text)
        # 去除多余空行
        text = re.sub(r'\n{3,}', '\n\n', text)
        # PDF 断行修复
        text = re.sub(r'(?<=[一-鿿])\n(?=[一-鿿])', '', text)
        return text.strip()
    
    def smart_split(self, text: str, chunk_size: int = 512, overlap: int = 64) -> list[dict]:
        """智能分块（参考 interview-agent 的 SmartTextSplitter）"""
        # 段落优先分割
        paragraphs = text.split('\n\n')
        
        chunks = []
        current_chunk = ""
        
        for para in paragraphs:
            if len(current_chunk) + len(para) <= chunk_size:
                current_chunk += para + "\n\n"
            else:
                if current_chunk:
                    chunks.append({"content": current_chunk.strip()})
                current_chunk = para + "\n\n"
        
        if current_chunk:
            chunks.append({"content": current_chunk.strip()})
        
        # 添加重叠窗口
        chunks = self.add_overlap(chunks, overlap)
        
        return chunks
```

#### 4.2.2 混合检索

**参考 interview-agent 的 HybridSearchService**：

```python
class HybridSearchService:
    """混合检索服务"""
    
    def __init__(self, vector_store, bm25_index, rrf_k=60):
        self.vector_store = vector_store
        self.bm25_index = bm25_index
        self.rrf_k = rrf_k
    
    async def search(self, query: str, domain: str = None, top_k: int = 20) -> list:
        """混合检索：向量 + 关键词 RRF 融合"""
        # 向量检索
        vector_results = await self.vector_search(query, domain, top_k)
        
        # 关键词检索（BM25）
        keyword_results = await self.keyword_search(query, domain, top_k)
        
        # RRF 融合
        fused = self.rrf_fusion(vector_results, keyword_results, top_k)
        
        return fused
    
    async def vector_search(self, query: str, domain: str, top_k: int) -> list:
        """向量语义检索"""
        embedding = self.embed_model.encode(query)
        results = self.vector_store.query(
            query_embeddings=[embedding],
            n_results=top_k,
            where={"domain": domain} if domain else None
        )
        return results
    
    async def keyword_search(self, query: str, domain: str, top_k: int) -> list:
        """BM25 关键词检索"""
        import jieba
        tokens = jieba.lcut(query)
        scores = self.bm25_index.get_scores(tokens)
        top_indices = scores.argsort()[-top_k:][::-1]
        return [self.documents[i] for i in top_indices]
    
    def rrf_fusion(self, vector_results, keyword_results, top_k) -> list:
        """Reciprocal Rank Fusion"""
        scores = {}
        
        for rank, doc in enumerate(vector_results):
            doc_id = doc["id"]
            scores[doc_id] = scores.get(doc_id, 0) + 1 / (self.rrf_k + rank + 1)
        
        for rank, doc in enumerate(keyword_results):
            doc_id = doc["id"]
            scores[doc_id] = scores.get(doc_id, 0) + 1 / (self.rrf_k + rank + 1)
        
        sorted_ids = sorted(scores.items(), key=lambda x: x[1], reverse=True)
        return [doc_id for doc_id, _ in sorted_ids[:top_k]]
```

#### 4.2.3 Re-ranking 精排

**参考 interview-agent 的 RerankingDocumentPostProcessor**：

```python
class Reranker:
    """Cross-Encoder Re-ranking 精排"""
    
    def __init__(self, model_name: str = "BAAI/bge-reranker-base"):
        from sentence_transformers import CrossEncoder
        self.model = CrossEncoder(model_name)
    
    async def rerank(self, query: str, candidates: list, top_k: int = 5) -> list:
        """Re-ranking 精排：Top-20 候选 → Top-5"""
        if not candidates:
            return []
        
        # 构建 query-document 对
        pairs = [[query, doc["content"]] for doc in candidates]
        
        # 计算相关性分数
        scores = self.model.predict(pairs)
        
        # 按分数排序，取 Top-K
        scored_docs = list(zip(candidates, scores))
        scored_docs.sort(key=lambda x: x[1], reverse=True)
        
        return [doc for doc, score in scored_docs[:top_k]]
```

```python
class KnowledgeSearchTool:
    """知识库检索工具（供 LLM 自主调用）"""
    
    name = "knowledge_search"
    description = "从知识库中检索相关文档，用于回答用户问题"
    
    def __init__(self, hybrid_search: HybridSearchService, reranker: Reranker):
        self.hybrid_search = hybrid_search
        self.reranker = reranker
    
    async def invoke(self, query: str, domain: str = None) -> str:
        """执行检索"""
        # 混合检索
        candidates = await self.hybrid_search.search(query, domain, top_k=20)
        
        # Re-ranking 精排
        reranked = await self.reranker.rerank(query, candidates, top_k=5)
        
        # 格式化结果
        results = []
        for i, doc in enumerate(reranked):
            results.append(f"[{i+1}] {doc['content'][:200]}...")
        
        return "\n\n".join(results)
```

#### 4.4.3 匹配引擎

```python
class MatchingEngine:
    """匹配引擎"""
    
    def __init__(self, embedding_model):
        self.embedding_model = embedding_model
    
    async def match(self, resume: dict, jd: dict) -> dict:
        """匹配简历与 JD"""
        # 1. 硬性条件检查
        hard_score, hard_details = self.check_hard_requirements(resume, jd)
        
        # 2. 技能语义匹配
        skill_score, skill_details = self.compute_skill_similarity(
            resume.get("skills", []),
            jd["hard_requirements"]["skills_required"]
        )
        
        # 3. 经验匹配
        exp_score, exp_details = self.compute_experience_match(
            resume.get("work_experience", []),
            jd
        )
        
        # 4. 项目匹配
        project_score, project_details = self.compute_project_match(
            resume.get("project_experience", []),
            jd
        )
        
        # 5. 综合评分（加权平均）
        final_score = (
            0.3 * hard_score +
            0.3 * skill_score +
            0.2 * exp_score +
            0.2 * project_score
        )
        
        return {
            "final_score": round(final_score, 4),
            "hard_requirements": {"score": hard_score, "details": hard_details},
            "skill_match": {"score": skill_score, "details": skill_details},
            "experience_match": {"score": exp_score, "details": exp_details},
            "project_match": {"score": project_score, "details": project_details}
        }
    
    def check_hard_requirements(self, resume: dict, jd: dict) -> tuple:
        """硬性条件检查"""
        scores = []
        details = {}
        
        # 学历检查
        resume_degree = self.get_highest_degree(resume.get("basic_info", {}).get("education", []))
        required_degree = jd["hard_requirements"]["education"]["min_degree"]
        degree_score = self.compare_degree(resume_degree, required_degree)
        scores.append(degree_score)
        details["education"] = {"resume": resume_degree, "required": required_degree, "match": degree_score}
        
        # 年限检查
        years = self.calculate_experience_years(resume.get("work_experience", []))
        min_years = jd["hard_requirements"]["experience"]["min_years"]
        max_years = jd["hard_requirements"]["experience"]["max_years"]
        years_score = 1.0 if min_years <= years <= max_years else max(0, 1 - abs(years - min_years) * 0.2)
        scores.append(years_score)
        details["experience_years"] = {"resume": years, "required": f"{min_years}-{max_years}", "match": years_score}
        
        return sum(scores) / len(scores), details
    
    def compute_skill_similarity(self, resume_skills: list, jd_skills: list) -> tuple:
        """技能语义匹配"""
        if not resume_skills or not jd_skills:
            return 0.0, {"matched": [], "missing": jd_skills}
        
        # Embedding 编码
        resume_embeddings = self.embedding_model.encode(resume_skills)
        jd_embeddings = self.embedding_model.encode(jd_skills)
        
        # 余弦相似度矩阵
        from sklearn.metrics.pairwise import cosine_similarity
        sim_matrix = cosine_similarity(resume_embeddings, jd_embeddings)
        
        # 每个 JD 技能找最匹配的简历技能
        matched = []
        missing = []
        for i, jd_skill in enumerate(jd_skills):
            max_sim = sim_matrix[:, i].max()
            if max_sim > 0.7:  # 相似度阈值
                best_match_idx = sim_matrix[:, i].argmax()
                matched.append({"jd_skill": jd_skill, "resume_skill": resume_skills[best_match_idx], "similarity": round(max_sim, 3)})
            else:
                missing.append(jd_skill)
        
        score = len(matched) / len(jd_skills)
        return score, {"matched": matched, "missing": missing}
```

### 4.5 RAG 评估模块

**参考 interview-agent 的 RAGAS 实现**：

```python
class RAGASEvaluator:
    """RAGAS 评估器"""
    
    async def evaluate(self, test_cases: list[dict]) -> dict:
        """
        评估 RAG 系统
        
        test_cases: [{"question": "...", "ground_truth": "...", "ground_truth_contexts": [...]}]
        """
        results = []
        
        for case in test_cases:
            # 1. 运行 RAG 获取回答
            answer, contexts = await self.run_rag(case["question"])
            
            # 2. 计算各指标
            faithfulness = await self.calc_faithfulness(answer, contexts)
            context_precision = await self.calc_context_precision(case["question"], contexts, case.get("ground_truth_contexts"))
            context_recall = await self.calc_context_recall(contexts, case.get("ground_truth_contexts"))
            answer_relevancy = await self.calc_answer_relevancy(case["question"], answer)
            
            results.append({
                "question": case["question"],
                "answer": answer,
                "metrics": {
                    "faithfulness": faithfulness,
                    "context_precision": context_precision,
                    "context_recall": context_recall,
                    "answer_relevancy": answer_relevancy
                }
            })
        
        # 汇总
        return {
            "faithfulness": np.mean([r["metrics"]["faithfulness"] for r in results]),
            "context_precision": np.mean([r["metrics"]["context_precision"] for r in results]),
            "context_recall": np.mean([r["metrics"]["context_recall"] for r in results]),
            "answer_relevancy": np.mean([r["metrics"]["answer_relevancy"] for r in results]),
            "details": results
        }
    
    async def calc_faithfulness(self, answer: str, contexts: list[str]) -> float:
        """忠实度：从回答提取声明 → 验证是否有上下文支持"""
        # Step 1: 提取声明
        claims = await self.extract_claims(answer)
        
        if not claims:
            return 1.0
        
        # Step 2: 验证每个声明
        supported_count = 0
        for claim in claims:
            is_supported = await self.verify_claim(claim, contexts)
            if is_supported:
                supported_count += 1
        
        return supported_count / len(claims)
    
    async def extract_claims(self, answer: str) -> list[str]:
        """从回答中提取事实性声明"""
        prompt = f"请从以下回答中提取最多10个核心事实性陈述，以JSON数组格式输出，不要添加其他文字：\n{answer}"
        result = await self.llm.ainvoke(prompt)
        try:
            return json.loads(result)
        except:
            return []
    
    async def verify_claim(self, claim: str, contexts: list[str]) -> bool:
        """验证声明是否有上下文支持"""
        context_block = "\n---\n".join(contexts)
        prompt = f"判断以下陈述是否被参考资料支持。只回答 SUPPORTED 或 NOT_SUPPORTED。\n\n参考资料：\n{context_block}\n\n待验证陈述：\n{claim}"
        result = await self.llm.ainvoke(prompt)
        return "SUPPORTED" in result.upper()
```

### 4.6 技能差距分析模块（扩展功能）

```python
class SkillGapAnalyzer:
    """技能差距分析器"""
    
    async def analyze(self, resume: dict, jd: dict) -> dict:
        """分析技能差距"""
        # 1. 提取技能集合
        resume_skills = set(resume.get("skills", []))
        required_skills = set(jd["hard_requirements"]["skills_required"])
        preferred_skills = set(jd["hard_requirements"].get("skills_preferred", []))
        
        # 2. 计算匹配情况
        matched_required = resume_skills & required_skills
        missing_required = required_skills - resume_skills
        matched_preferred = resume_skills & preferred_skills
        missing_preferred = preferred_skills - resume_skills
        
        # 3. 生成学习推荐
        recommendations = await self.generate_recommendations(missing_required, missing_preferred)
        
        # 4. 计算匹配率
        required_match_rate = len(matched_required) / len(required_skills) if required_skills else 1.0
        preferred_match_rate = len(matched_preferred) / len(preferred_skills) if preferred_skills else 1.0
        
        return {
            "required_match_rate": round(required_match_rate, 3),
            "preferred_match_rate": round(preferred_match_rate, 3),
            "overall_match_rate": round((required_match_rate + preferred_match_rate) / 2, 3),
            "matched_required": list(matched_required),
            "missing_required": list(missing_required),
            "matched_preferred": list(matched_preferred),
            "missing_preferred": list(missing_preferred),
            "recommendations": recommendations
        }
    
    async def generate_recommendations(self, missing_required: set, missing_preferred: set) -> list:
        """生成学习推荐"""
        if not missing_required and not missing_preferred:
            return [{"message": "恭喜！你已满足所有技能要求"}]
        
        prompt = f"""作为职业顾问，请为以下缺失技能提供学习推荐。

必须掌握的技能（缺失）：{', '.join(missing_required) if missing_required else '无'}
加分技能（缺失）：{', '.join(missing_preferred) if missing_preferred else '无'}

请为每个技能提供：
1. 推荐学习资源（书籍/课程/文档）
2. 预计学习时间
3. 学习优先级（高/中/低）

以JSON数组格式输出。"""
        
        result = await self.llm.ainvoke(prompt)
        try:
            return json.loads(result)
        except:
            return [{"skill": s, "priority": "高"} for s in missing_required]
```
